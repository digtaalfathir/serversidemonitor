🏗️ Digital Twin

MONITORING v2 · BERDIRI SENDIRI
Aplikasi v2 terpisah dari app monitoring. Status device live diambil dari backend monitoring lewat WebSocket (proxy).
🏗️
Builder (3D / 2D)
Satu app — tinggal pilih 3D atau 2D. Bikin denah → simpan scene.json / layout2d.json
🛰️
Viewer 3D
Muat scene.json + status device live (hijau/merah by-IP)
🗺️
Viewer 2D
Peta 2D top-down (SVG) dari layout2d.json + status live
Jalankan: npm run v2 (port 10102) · Status device di-proxy dari backend monitoring (npm start, port 10101).
Belum ada scene? Buka Builder → pilih 3D/2D → Simpan → taruh scene.json / layout2d.json di v2/public/. Contoh: 3D · 2D.
