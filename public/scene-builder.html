Tembok

Klik titik demi titik di lantai untuk membentuk tembok. Enter/klik-dobel = selesai. Esc = batal. Aktifkan "tutup loop" untuk tembok luar keliling.

Lantai

Klik dua titik (sudut ke sudut) untuk membuat kotak lantai. Buat beberapa area dengan tipe berbeda (mis. jalur hijau + beton).

Pin Device

Klik di lantai untuk menaruh titik status. Pin ber-IP akan menyala hijau/merah sesuai status live di dashboard.
